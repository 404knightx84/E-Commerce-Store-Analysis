# 📊 E-Commerce Store Analysis Dashboard

A comprehensive Power BI dashboard designed to analyze and visualize e-commerce store performance metrics. This project demonstrates advanced business intelligence techniques for tracking sales data, customer behavior, and operational insights.

## 🎯 Project Overview

This repository contains an interactive Power BI dashboard that provides deep insights into online sales performance. It serves as an excellent learning resource for anyone interested in data visualization and business intelligence with Power BI. The dashboard connects to real sales data and presents key metrics through multiple visualization types.

## ✨ Features

- **Interactive Dashboard**: Drill-down capabilities and dynamic filtering for detailed data exploration
- **Advanced Visualizations**: Multiple chart types including bar, pie, donut, scatter, line, and area charts
- **Custom Slicers**: User-driven parameters for real-time data filtering
- **Data Integration**: Connected tables with calculated fields and aggregations
- **Geographic Mapping**: Sales performance visualization across regions
- **Real-time Insights**: Performance metrics that update based on selected filters

## 📦 Requirements

- **Microsoft Power BI Desktop** (Latest version recommended)
  - [Download Power BI Desktop](https://powerbi.microsoft.com/en-us/downloads/)
- Windows 10 or later (or Power BI service on any OS)

## 💾 Installation

1. **Clone or download** this repository to your local machine
2. **Install Power BI Desktop** if you haven't already
3. **Open** `E-commerce Sales Dashboard.pbix` with Power BI Desktop
4. The dashboard will automatically load the data from the CSV files in the `dataset/` folder

## 📂 Data Structure

The project includes two main data sources:

### dataset/
- **Orders.csv** - Order transactions, dates, and quantities
- **Details.csv** - Product details, pricing, and order line items

These files are connected within the Power BI file to create a unified data model.

## 🚀 How to Use

1. **Open the Dashboard**: Launch `E-commerce Sales Dashboard.pbix` in Power BI Desktop
2. **Use Slicers**: Click on slicer elements to filter data by date, region, product, etc.
3. **Drill Down**: Click on chart elements to dive deeper into specific data segments
4. **Explore Metrics**: Hover over visualizations to see detailed information
5. **Export Reports**: Generate reports or export data for presentations
